# MNIST-ensemble-and-stacking-investigation
This is my attempt at exercises 8 and 9 from chapter 6 of Hands-on Machine Learning


This project explores whether combining multiple classifiers improves handwritten
digit recognition on the MNIST dataset.

The notebook compares:

Random Forest
Extra Trees
RBF Support Vector Classifier (SVC)
Majority-vote ensemble
Logistic regression as a stacking blender
StackingClassifier from scikit-learn
